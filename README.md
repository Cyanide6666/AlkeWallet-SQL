# Alke Wallet - Base de Datos Relacional
Proyecto que implementa el backend de una billetera digital con soporte para múltiples divisas y gestión de usuarios.

# Características Técnicas
Integridad Referencial: Uso de llaves primarias y foráneas

Multimoneda: Sistema capaz de registrar movimientos en CLP y USD de forma dinámica.

# Stack Tecnológico
Motor de Base de Datos: PostgreSQL.

Herramienta de Gestión: DBeaver.

Lenguaje: SQL (DDL, DML, DQL).
